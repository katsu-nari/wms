## 主な変更点（既存 `wms_supabase.html` からのスコープ変更）

- ❌ **削除**: クライアント（荷主）管理機能
- ➕ **追加**: 棚卸機能（指示→カウント→差異→確定）
- ➕ **追加**: ロケーション移動機能
- ➕ **追加**: ユーザー認証（**社員番号 + 数字5桁PIN**）・ロールベース権限管理
- ➕ **追加**: ログイン試行レート制限 / アカウントロック（5桁PINの脆弱性補強）
- ➕ **追加**: CSV/帳票エクスポート

## 次のステップ

実装フェーズ（P1 以降）では、本ドキュメント群を参照しながら
以下の順で構築を進めます。

1. Supabase マイグレーション（`supabase/migrations/`）
2. 認証・マスタ・ロケーション・在庫一覧
3. 入庫・出庫フロー（スキャン含む）
4. ロケーション移動・棚卸
5. 帳票・CSV・モバイル最適化
